## データについて

本分析で利用したデータは、`data/apparel.xlsx`に格納されています。

### データ形式

| ds         | Weather | Bargain | y      |
|------------|---------|---------|--------|
| 1/5/1990   | 0       | 0       | 58,507 |
| 1/6/1990   | 0       | 0       | 49,088 |
| 1/7/1990   | 0       | 0       | 50,066 |
| 1/8/1990   | 0       | 0       | 45,954 |

### カラム説明

| カラム名 | 説明 |
|----------|------|
| **ds** | 日付（売上日） |
| **Weather** | 天候フラグ（0=晴れ, 1=雨） |
| **Bargain** | バーゲン施策フラグ（0=通常, 1=バーゲン実施） |
| **y** | 売上金額（円） |

> **注意**: 休業日のデータは欠損値として記録されています。

## 分析手法

本プロジェクトでは、**Facebook Prophet** を使用した状態空間モデルによる時系列予測を実施しています。

### 分析のステップ

1. **データの前処理**
   - 日付型への変換
   - 売上金額の対数変換

2. **トレンド分析**
   - 長期的なトレンドの抽出
   - 週次・年次の周期性の分析

3. **外部要因の考慮**
   - 天候効果（雨の日の影響）
   - バーゲン効果（セール実施の影響）
   - 祝日効果（日本の祝日の影響）

4. **需要予測**
   - 1年先（365日）の売上予測
